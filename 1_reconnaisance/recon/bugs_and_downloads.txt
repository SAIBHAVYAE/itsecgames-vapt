    <h1>Bugs</h1>

    <p>What makes bWAPP, our extremely buggy web application, so unique? Well, it has over <a href="./downloads/vulnerabilities.txt" target="blank">100 web bugs</a>!<br />
    bWAPP covers all vulnerabilities from the OWASP Top 10 project, including:

    <ul>

        <li>SQL, HTML, iFrame, SSI, OS Command, PHP, XML, XPath, LDAP, Host Header and SMTP injections</li>
        <li>Cross-Site Scripting (XSS), Cross-Site Tracing (XST) and Cross-Site Request Forgery (CSRF)</li>
        <li>AJAX and Web Services issues (jQuery/JSON/XML/SOAP/WSDL)</li>
        <li>Authentication, authorization and session issues, file upload flaws and backdoor files</li>
        <li>Arbitrary file access, directory traversals, local and remote file inclusions (LFI/RFI)</li>
        <li>Configuration issues: Man-in-the-Middle, cross-domain policy files, information disclosures,...</li>
        <li>HTTP parameter pollution, HTTP response splitting and HTTP verb tampering</li>
        <li>Insecure DistCC, FTP, NTP, Samba, SNMP, VNC and WebDAV configurations</li>
        <li>HTML5 ClickJacking, Cross-Origin Resource Sharing (CORS) and web storage issues</li>
        <li>XML External Entity attacks (XXE) and Server Side Request Forgery (SSRF)</li>
  		<li>Heartbleed and Shellshock vulnerability (OpenSSL), Denial-of-Service (DoS) attacks</li>
		<li>Parameter tampering, cookie and password reset poisoning</li>
